Descriptive Statistics¶

The descriptive_stats component extracts a number of descriptive statistics. The following attributes are added:

  • ._.counts (Doc & Span)

    • Number of tokens.

    • Number of unique tokens.

    • Proportion unique tokens.

    • Number of characters.

  • ._.sentence_length(Doc)

    • Mean sentence length.

    • Median sentence length.

    • Std of sentence length.

  • ._.syllables(Doc)

    • Mean number of syllables per token.

    • Median number of syllables per token.

    • Std of number of syllables per token.

  • ._.token_length(Doc & Span)

    • Mean token length.

    • Median token length.

    • Std of token length.

spaCy v.3.0 component that adds attributes with desriptive statistics to Doc and Span objects. The attributes relate to token and sentence length, number of syllables, and counts of tokens and sentences.

textdescriptives.components.descriptive_stats.create_descriptive_stats_component(nlp: spacy.language.Language, name: str)[source]¶

Allows DescriptiveStatistics to be added to a spaCy pipe using nlp.add_pipe(“descriptive_stats”). If the pipe does not contain a parser or sentencizer, the sentencizer component is silently added.
