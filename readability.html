Readability¶

The readability component adds the following readabiltiy metrics under the ._.readability attribute to Doc objects.

  • Gunning-Fog

  • SMOG

  • Flesch reading ease

  • Flesch-Kincaid grade

  • Automated readability index

  • Coleman-Liau index

  • Lix

  • Rix

For specifics of the implementation, refer to the source. The equations are largely derived from the textstat library.

spaCy v.3.0 component for adding readability metrics to Doc objects. Extracts metrics and returns them as a dictionary as the ._.readability attribute.

score = 0.0588 * avg number of chars pr 100 words - 0.296 * avg num of sents pr 100 words -15.8

Score = grade required to read the text

_flesch_kincaid_grade(doc: spacy.tokens.doc.Doc)[source]¶

Score = grade required to read the text

_flesch_reading_ease(doc: spacy.tokens.doc.Doc)[source]¶

206.835 - (1.015 X avg sent len) - (84.6 * avg_syl_per_word)

Higher = easier to read

Works best for English

_gunning_fog(doc, hard_words: int)[source]¶

Grade level = 0.4 * ((avg_sentence_length) + (percentage hard words))

hard words = 3+ syllables

_lix(doc: spacy.tokens.doc.Doc, long_words: int)[source]¶

(n_words / n_sentences) + (n_words longer than 6 letters * 100) / n_words

_rix(doc: spacy.tokens.doc.Doc, long_words: int)[source]¶

n_long_words / n_sentences

_smog(doc: spacy.tokens.doc.Doc, hard_words: int)[source]¶

grade level = 1.043( sqrt(30 * (hard words /n sentences)) + 3.1291

Preferably need 30+ sentences. Will not work with less than 4

textdescriptives.components.readability.create_readability_component(nlp: spacy.language.Language, name: str)[source]¶

Allows Readability to added to a spaCy pipe using nlp.add_pipe(“readability”). Readability requires attributes from DescriptiveStatistics and adds it to the pipe if it not already loaded.
